# PalmMemo

A learning tool for Android, iOS and Web, inspired by palm sugarmemo, write in flutter.

## Web Shortcuts
- Enter: next word
- Space: show word meaning
- Minus: speak word(baidu)
- Equal: speak word(google)
- Del: delete word
